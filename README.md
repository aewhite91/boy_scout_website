# My Website Project

This is a simple static website hosted using GitHub Pages.

---

## 🧭 Purpose

This project was created as part of my step-son’s Boy Scouts project. It is intended as a hands-on introduction to basic web development and modern software tools.

Through this project, he learns how to:
- Build a basic website using HTML and CSS
- Organize files in a project structure
- Use Git for version control
- Host a live website using GitHub Pages

---

## 🌐 What This Site Is

This is a very simple static website built using only:

- `index.html`
- `styles.css`

There are no frameworks, build tools, or backend services involved. The goal is to keep everything simple and easy to understand.

---

## ⚙️ How It Works

GitHub Pages automatically publishes this repository as a static website.

Any changes pushed to the `main` branch will automatically update the live site.

---

## 📁 Project Structure
├── index.html # Main webpage content

├── styles.css # Styling for the site

└── .gitignore # Files excluded from version control


---

## 🎯 Learning Goals

This project demonstrates:

- Basic web development (HTML/CSS)
- File organization in a real project
- Using Git and GitHub for version control
- Publishing a live website for free using GitHub Pages

---

## 🚀 Live Site

Once GitHub Pages is enabled, the site will be available at: https://<your-username>.github.io/<repo-name>
